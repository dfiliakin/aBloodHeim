# aBloodHeim

## Setup
Setting up Python Virtual Environment
```
python -m venv .venv
source .venv/Scripts/activate
```

Setting up Node.js with TypeScript
```
npm install
```

## Client
Scripts
```
npm test
npm build
npm start:dev
npm start
```

## Server
uvicorn main:app --host 0.0.0.0 --port 8080 --reload
