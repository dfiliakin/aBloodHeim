# aBloodHeim

## Setup
Setting up Python Virtual Environment
```
python -m venv .venv
source .venv/Scripts/activate
```

Setting up Node.js with TypeScript
```
npm init -y
npm install typescript --save-dev
npm install @types/node --save-dev
```

## Client
Scripts
```
npm run test
npm run build
npm run start:dev
npm run start
```
